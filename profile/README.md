# 취업 관리 서비스
채용 공고 열람, 포트폴리오 작성, 회원 관리를 제공하는 취업 지원 웹 서비스입니다.

<br/><br/>

## 팀원
| 구분 | 이름 | 역할 |
|---|---|---|
| 팀장 | 조은솔 | 프론트엔드, 개발 환경 구축, Nginx 서버 구축 |
| 팀원 | 김도연 | DB 설계 및 구축, 백엔드 |
| 팀원 | 박준호 | 프론트엔드 |
| 팀원 | 이석진 | 크롤링, 백엔드 |
| 팀원 | 이승재 | 백엔드, fastAPI 서버 구축, 디스코드 알림 |

<br/><br/>

## 서비스 구성
- 3-tier 구축(WEB-WAS-DB)
    - WEB: Nginx
    - WAS: FastAPI
    - DB: MariaDB
- 통신 프로토콜: HTTP

<br/><br/>

## 개발 기간 및 환경
- 개발 기간: 2026.07.08 ~ 2026.07.16
- 개발 환경: Raspberry Pi 2B
- 운영체제: Ubuntu 22.04.5
- 사용 언어: Python 3.12
